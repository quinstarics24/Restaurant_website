# Aunty Co's Kitchen

Aunty Co's Kitchen is a responsive and dynamic restaurant website showcasing traditional Cameroonian dishes. It features a daily-updated menu, online reservation system, image gallery, and contact information. The website includes an admin dashboard for managing daily meals, weekly gallery updates, and viewing customer reservations.

## Features

- Responsive design compatible with desktop and mobile devices
- Daily-updated menu showcasing 3 traditional Cameroonian meals
- Online reservation form for customers to book tables easily
- Image gallery highlighting the restaurant’s dishes and ambiance
- Contact page with phone, WhatsApp, and location details
- Secure admin dashboard to:
  - Update daily meals (name, description, price, image)
  - Upload and manage gallery images weekly
  - View and manage customer reservations

## Technologies Used

- HTML5 & CSS3 for structure and styling
- JavaScript for frontend interactivity
- PHP for backend processing and admin panel
- MySQL for database management
- XAMPP as the local development server
